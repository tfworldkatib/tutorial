Now, let us connect to Katib UI. To do this we forward the Katib UI service to a port on the Vagrant VM as follows.

kubectl -n kubeflow patch service  katib-ui \
    --patch='{"spec":{"$setElementOrder/ports":[{"port":80}],"ports":[{"nodePort":31230,"port":80}],"type":"NodePort"}}'

Check that the Katib UI service shows as a NodePort service.

kubectl -n kubeflow get svc katib-ui
Sample Output

NAME TYPE CLUSTER-IP EXTERNAL-IP PORT(S) AGE katib-ui NodePort 10.99.22.59 80:31230/TCP 10h
